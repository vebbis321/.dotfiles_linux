# NVIM

## netrw

```lua
"n", "<leader>fe"
d: new folder
%: new file
```

## telescope

```lua
"n", "<C-p>"
"n", "<leader>pf"
"n", "<leader>sf"
```

## harpoon

```lua
"n", "<leader>a", # append
"n", "<C-e>", # open meny
"n", "<C-a>", # go to 1
"n", "<C-s>", # go to 2
```

## nvim-cmp

```lua
<CR> # completion
<C-k> # previous
<C-j> # next
<C-c> # cancel
```

## lspconfig

```lua
"n", "<gd>", # jump to definition
"n", "K",  # hover
```
