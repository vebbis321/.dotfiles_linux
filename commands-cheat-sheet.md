# NVIM

## netrw

```lua
"n", "<leader>fe"
d: new folder
%: new file
```

## remaps

```lua
"n", "<leader>s", # replace on current word
"n", "<leader>x", # CHMOOOD
```

## telescope

```lua
"n", "<C-p>"
"n", "<leader>pf"
"n", "<leader>sf"
```

## harpoon

```lua
"n", "<leader>a", # append
"n", "<C-e>", # open meny
"n", "<C-h>", # go to 1
"n", "<C-j>", # go to 2
```

## nvim-cmp

```lua
<CR> # completion
<C-k> # previous
<C-j> # next
<C-c> # cancel
```

## lspconfig

```lua
"n", "<gd>", # jump to definition
"n", "K",  # hover
```
